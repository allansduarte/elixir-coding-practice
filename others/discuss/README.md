# Discuss

Simple CRUD application.

## Project setup

To get this project working is needed to run the migrations and install the dependencies:

```
$ mix ecto.setup
$ mix deps.get
$ mix deps.compile
```

## Running the project

To get this project working is needed to run the migrations and install the dependencies:

```
$ mix phoenix.server
```
